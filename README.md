# WB_TEST_TASK_1

## 📝 Описание задания
Целью данного тестового задания являлся анализ $20 000$ поисковых запросов пользователей маркетплейса. Решение разделено на несколько ключевых этапов:

1. **Разведочный анализ данных (EDA):** Загрузка данных, анализ статистик, распределений длин запросов, частотности токенов. Проведена векторизация и кластеризация текстов для извлечения скрытых паттернов и типичных сценариев поиска;
2. **Извлечение атрибутов (Rule-based):** Разработка Python-функций для извлечения сущностей (бренды, категории, атрибуты) из текстов запросов БЕЗ использования тяжелых ML/NLP-моделей. Использовались регулярные выражения, словари и строковые методы;
3. **Анализ и оценка:** Описание типичных ошибок, разбор сложных и двусмысленных запросов (опечатки, сленг, составные сущности), оценка качества rule-based решений и предложение дальнейших шагов по улучшению (переход к NER-моделям).

## ⚙️ Установка

1. **Клонируйте репозиторий**:
```bash

cd WB_TEST_TASK_1
```
2. **Установите зависимости**:
```bash

```
